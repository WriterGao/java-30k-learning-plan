# Day11：设计模式（创建型）

本目录用于存放「第 11 天」的学习资料、实验代码与当天笔记产出。

## 今日计划（来自总计划）

- [ ] 理解设计模式的概念与 GOF 23 种设计模式分类
- [ ] 掌握 7 大设计原则（单一职责/开闭/里氏替换/依赖倒置/接口隔离/迪米特/合成复用）
- [ ] 深入掌握单例模式的 6+ 种实现方式及其优缺点
- [ ] 理解 DCL 双重检查锁为什么需要 volatile（指令重排序）
- [ ] 掌握反射破坏单例与防御方案、序列化破坏单例与 readResolve() 防御
- [ ] 理解简单工厂、工厂方法、抽象工厂三种工厂模式的区别与适用场景
- [ ] 掌握建造者模式（经典建造者 + 链式建造者）
- [ ] 理解原型模式（Cloneable + 浅拷贝 vs 深拷贝）
- [ ] 了解创建型模式在 Spring / JDK 中的实际应用
- [ ] 编写实验代码验证各知识点
- [ ] 写学习笔记，总结设计模式创建型模式
