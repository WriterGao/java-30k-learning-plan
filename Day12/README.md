# Day12：设计模式（结构型）

本目录用于存放「第 12 天」的学习资料、实验代码与当天笔记产出。

## 今日计划（来自总计划）

- [ ] 理解结构型模式的概念与核心思想（组合优于继承）
- [ ] 掌握适配器模式（对象适配器、类适配器、接口适配器）
- [ ] 理解桥接模式（抽象与实现分离，避免类爆炸）
- [ ] 掌握组合模式（树形结构，统一处理单个对象和组合对象）
- [ ] 理解装饰器模式（动态扩展功能，避免类爆炸）
- [ ] 掌握外观模式（简化复杂子系统接口）
- [ ] 理解享元模式（共享细粒度对象，节省内存）
- [ ] 深入掌握代理模式（静态代理、JDK 动态代理、CGLIB）
- [ ] 理解 JDK 动态代理和 CGLIB 的区别与选择
- [ ] 了解结构型模式在 Spring / JDK 中的实际应用
- [ ] 编写实验代码验证各知识点
- [ ] 写学习笔记，总结设计模式结构型模式

## 📚 推荐学习路径

**🎯 最佳学习方式**：直接使用 `00-教学课件_设计模式之结构型模式详解.md`（导师定制，图文并茂，Mermaid 类图辅助理解）

3. **第三步**：完成 `03-实验与练习.md` 中的实验
4. **第四步**：在 `02-学习笔记.md` 中记录你的理解和思考
