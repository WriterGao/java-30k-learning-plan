# Day12：设计模式（结构型）

本目录用于存放「第 12 天」的学习资料、实验代码与当天笔记产出。

## 今日计划（来自总计划）

- [ ] 理解结构型设计模式的核心思想（通过组合获得更大的结构）
- [ ] 掌握适配器模式（类适配器、对象适配器）
- [ ] 掌握装饰器模式（与继承的区别、与代理模式的区别）
- [ ] 深入理解代理模式（静态代理、JDK动态代理、CGLIB动态代理）
- [ ] 理解外观模式（简化复杂子系统接口）
- [ ] 理解桥接模式（抽象与实现分离）
- [ ] 了解组合模式、享元模式
- [ ] 理解结构型模式在 Spring / JDK 中的实际应用
- [ ] 编写实验代码验证各知识点
- [ ] 写学习笔记，总结结构型设计模式
