# Day03：JVM 调优实战

本目录用于存放「第 3 天」的学习资料、实验代码与当天笔记产出。

## 今日学习目标

- ✅ 掌握 JVM 参数体系（标准参数、-X参数、-XX参数）
- ✅ 掌握堆内存调优（-Xms/-Xmx/-Xmn/NewRatio/SurvivorRatio）
- ✅ 理解 GC 调优策略（吞吐量优先 vs 低延迟优先）
- ✅ 学会阅读和解析 GC 日志
- ✅ 熟练使用 JVM 监控工具（jps/jstat/jinfo/jmap/jstack/jcmd）
- ✅ 了解图形化工具（jconsole/jvisualvm/Arthas/MAT）的使用
- ✅ 掌握常见 OOM 问题排查流程
- ✅ 掌握线上问题排查方法（CPU 飙高、内存泄漏、频繁 Full GC）
- ✅ 能对 Spring Boot 项目进行 JVM 调优

> **前置知识**：Day01（JVM 内存模型）、Day02（GC 算法与垃圾收集器）

## 📚 推荐学习路径

1. **第一步**：阅读 `00-教学课件_JVM调优实战详解.md`（核心课件，必读）
2. **第二步**：运行 `code/` 目录下的实验代码，复现常见问题
3. **第三步**：按照 `03-实验与练习.md` 完成动手实验
4. **第四步**：在 `02-学习笔记.md` 中记录你的理解和思考
