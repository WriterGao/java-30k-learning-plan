# 书籍资源目录

本目录用于存放《深入理解Java虚拟机（第3版）》的电子版资源。

## ⚠️ 版权说明

**本仓库不会提供受版权保护的书籍原文下载链接**。请通过合法渠道获取书籍。

## 📚 合法获取途径

### 1. 购买正版电子书
- **京东读书**：https://e.jd.com/
- **当当云阅读**：https://e.dangdang.com/
- **微信读书**：https://weread.qq.com/
- **Kindle**：Amazon Kindle商店
- **Apple Books**：App Store

### 2. 购买纸质书
- **京东**：搜索"深入理解Java虚拟机 第3版 周志明"
- **当当**：搜索"深入理解Java虚拟机 第3版"
- **机械工业出版社官方渠道**

### 3. 图书馆借阅
- 公共图书馆
- 高校图书馆
- 部分图书馆提供电子版借阅服务

## 📁 如果你已有合法电子版

如果你已经通过合法途径获取了电子版（PDF、EPUB等），可以：

1. 将文件复制到本目录（`Day01/resources/`）
2. 建议命名格式：`深入理解Java虚拟机-第3版-周志明.pdf`
3. 注意：**不要将受版权保护的文件推送到公开的GitHub仓库**（可能违反版权法）

## 🔒 私有仓库建议

如果你需要将书籍文件放入项目：
- 建议使用**私有Git仓库**（GitHub Private Repository）
- 或者使用 `.gitignore` 排除书籍文件，仅本地使用
