# 版本确认（避免“原版搞错”）

你要求“**一定要原版，不要搞错**”，并且希望我“**找最新的一版**”。因此本仓库默认以《深入理解 Java 虚拟机》的**最新版（通常为第 3 版）**作为学习基准；但为了避免“版次/印次/封面相似导致拿错书”，仍然请你在这里补全**你实际使用的那本**的信息（以书的版权页为准）。
