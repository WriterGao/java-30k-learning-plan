# Day01：JVM 内存模型基础

本目录用于存放「第 1 天」的学习资料、实验代码与当天笔记产出。

## 今日计划（来自总计划）

- 学习 JVM 内存结构（堆、栈、方法区、程序计数器）

- 学习对象创建过程
- 阅读《深入理解 Java 虚拟机》第 2 章
- 编写代码验证堆内存分配
- 使用 jvisualvm/VisualVM 观察内存使用情况
- 练习：创建大量对象观察 GC 情况
- 写学习笔记，总结 JVM 内存模型
